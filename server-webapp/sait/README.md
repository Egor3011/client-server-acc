# ACC Server WebApp

Frontend для управления серверами:
- на главной странице показывает список серверов из backend;
- при переходе в карточку сервера открывает страницу управления;
- на странице сервера подключается к `ws://<ip>:8765` и отправляет команды `start`, `stop`, `restart`.

## Запуск

```sh
npm install
npm run dev
```

## Переменные окружения (опционально)

- `VITE_API_BASE_URL` — адрес backend API (по умолчанию `http://127.0.0.1:8000`)
- `VITE_SERVER_WS_PORT` — порт WebSocket-сервера (по умолчанию `8765`)

## Проверка сборки

```sh
npm run build
```
